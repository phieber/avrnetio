Welcome to avrnetio

This is a software project written in Python to make interaction with the ethersex platform really easy for you. You can query Data via . See ethersex.deI for details about the ethersex project. In contrast to what the project name avrnetio might suppose, you can interact with any ethersex hardware, not just the commercial Pollin AVR-NET-IO.
